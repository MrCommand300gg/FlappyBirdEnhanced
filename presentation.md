# Презентация проекта

## 1. Название

**Flappy Bird Enhanced** — улучшенная версия игры Flappy Bird

## 2. Цель и задачи проекта

**Цель:** Разработать полнофункциональную мобильную игру на платформе Android с использованием фреймворка libGDX, включающую классический геймплей Flappy Bird с расширенными возможностями.

**Задачи:**
- Реализовать физику птицы с гравитацией и вращением
- Создать систему динамического уровня сложности
- Добавить коллекционные элементы (звёзды) и бонусы
- Разработать систему достижений и таблицу лидеров
- Реализовать полноценное меню, настройки и профиль игрока
- Добавить визуальные и звуковые эффекты
- Обеспечить адаптацию под различные экраны

## 3. Технологическая основа проекта

| Компонент | Технология |
|-----------|------------|
| Язык программирования | Java |
| Фреймворк | libGDX 1.12.0 |
| Сборка | Gradle |
| Платформы | Android, Desktop |
| Разрешение | 1280x720 (с адаптацией) |
| Минимальная версия Android | API 21 |

## 4. Реализованный функционал

| Категория | Функционал | Описание |
|-----------|------------|----------|
| **Геймплей** | Физика птицы | Гравитация, прыжки, вращение при движении |
| | Система труб | Динамическое появление с разными зазорами |
| | Коллекционные звёзды | +5 очков за каждую собранную звезду |
| | Система бонусов | Щит (5 сек), Ускорение (3 сек), Уменьшение (4 сек), Магнит (5 сек) |
| **Сложность** | Динамическая система | 4 уровня: Easy, Medium, Hard, Insane |
| | Адаптивная скорость | Увеличивается с ростом очков |
| **Интерфейс** | Главное меню | Навигация по играм |
| | Экран настроек | Громкость музыки и эффектов, выбор сложности |
| | Экран паузы | Приостановка игры |
| | Экран завершения | Статистика, лучший счёт |
| | Обучение | Пошаговое руководство для новичков |
| **Системы** | Достижения | 7 достижений (First Flight, Novice, Experienced, Master, Legend, Collector, Dedicated) |
| | Таблица лидеров | Топ-10 лучших результатов |
| | Профиль игрока | Статистика и имя игрока |
| | Сохранение | Preferences API |
| **Визуал** | Параллакс фон | 4 слоя с разной скоростью |
| | Частицы | При прыжке, сборе звёзд, смерти |
| | Screen shake | Эффект тряски при столкновении |
| **Аудио** | Звуковые эффекты | Прыжок, сбор, удар, смерть, достижение |
| | Фоновая музыка | Меню и игра |

## 5. Перспективы развития проекта

- **Онлайн-таблица лидеров** — синхронизация с сервером для глобального рейтинга
- **Скинты для птицы** — разблокируемые визуальные темы
- **Ежедневные задания** — дополнительные цели для игроков
- **Мультиплеер** — соревнование с другими игроками в реальном времени
- **Новые уровни** — различные темы и препятствия
- **Рекламная интеграция** — монетизация для Android-версии
- **Облачные сохранения** — синхронизация прогресса между устройствами
